Set absolute descendants for this flow.

Set yourself as the Containing Block for all the absolute descendants.

This is called during flow construction, so nothing else can be accessing the descendant flows. This is enforced by the fact that we have a mutable FlowRef, which only flow construction is allowed to possess.
