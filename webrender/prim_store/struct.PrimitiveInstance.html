pub struct PrimitiveInstance {
    pub kind: PrimitiveInstanceKind,
    pub local_clip_rect: LayoutRect,
    pub id: PrimitiveDebugId,
    pub prepared_frame_id: FrameId,
    pub visibility_info: PrimitiveVisibilityIndex,
    pub clip_chain_id: ClipChainId,
}

Fields

kind: PrimitiveInstanceKind

Identifies the kind of primitive this instance is, and references to where the relevant information for the primitive can be found.

local_clip_rect: LayoutRect

Local space clip rect for this instance

id: PrimitiveDebugIdprepared_frame_id: FrameId

The last frame ID (of the RenderTaskGraph) this primitive was prepared for rendering in.

visibility_info: PrimitiveVisibilityIndex

If this primitive is visible, an index into the instance visibility scratch buffer. If not visible, INVALID.

clip_chain_id: ClipChainId

ID of the clip chain that this primitive is clipped by.
