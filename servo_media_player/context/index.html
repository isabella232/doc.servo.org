servo_media_player

[−][src]Module servo_media_player::context

PlayerGLContext is a trait to be used to pass the GL context for rendering purposes.

The current consumer of this trait is the GL rendering mechanism for the GStreamer backend.

The client application should implement this trait and pass the trait object to its player instance.
