httparse::simd

[−][src]Module httparse::simd::runtime

Runtime detection of simd features. Used when the build script doesn't notice any target features at build time.

While is_x86_feature_detected! has it's own caching built-in, at least in 1.27.0, the functions don't inline, leaving using it actually slower than just using the scalar fallback.
