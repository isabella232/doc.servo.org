Helper trait that describes what is expected from a GetSockOpt getter.

Required methods

unsafe fn blank() -> Self

Returns an empty value.

fn ffi_ptr(&mut self) -> *mut c_void

Returns a pointer to the stored value. This pointer will be passed to the system's getsockopt call (man 3p getsockopt, argument option_value).

fn ffi_len(&mut self) -> *mut socklen_t

Returns length of the stored value. This pointer will be passed to the system's getsockopt call (man 3p getsockopt, argument option_len).

Returns the stored value.
